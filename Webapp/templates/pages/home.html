{{_('Gameplay Analysis and Preservation')}}

{{_('This project aims at extracting gameplay data from game recordings.')}}

{{_('The goals of this project are :')}}

  • Provide a deep learning model able to recover the key logs from a simple video input
  • Provide a recording script, in the form of an OBS plugin, simplifying the recording of videos along with its key logs
  • Provide a report that explains the followed steps leading to our results
  • Provide an online analysis tool, running the deep learning model, and showing the game loops found inside the key logs
